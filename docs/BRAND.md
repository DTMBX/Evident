# Evident Brand & Design System

## Brand Overview

Evident is a legal-grade evidence processing platform focused on forensic clarity, auditability, and local-first privacy. The brand tone is precise, accessible, and trustworthyâ€”never playful or ambiguous.

- **Brand Name:** Evident
- **Tagline:** Evidence made legible. Truth made durable.
- **Core Values:** Forensic clarity, auditability, privacy, accessibility
- **Primary Color:** `--brand: #2F5D9F;` (Evident Blue)
- **Accent Color:** `--accent: #F7B32B;` (Gold)
- **Surface:** `--paper: #F8FAFC;` (Paper White)
- **Ink:** `--ink: #0B0D10;` (Near Black)
- **Muted:** `--muted: #6B7280;` (Muted Gray)
- **Pattern:** Subtle grid/pattern backgrounds for structure
- **Typography:** Modern, accessible sans-serif (see tokens.css)
- **Logo:** Scales of Justice, geometric, blue/gold

## Design Tokens (see assets/css/tokens.css)

- All colors, radii, shadows, spacing, and typography are defined as CSS variables in `assets/css/tokens.css`.
- Use only tokens for all UI elements.

## UI System

- **Buttons:** Unified `.btn` system (primary, secondary, ghost), pill radius, accessible focus, no gradients
- **Layout:** `.container`, `.grid`, `.card`, `.panel`, `.bg-grid` for responsive, accessible layouts
- **Patterns:** Subtle SVG/CSS grid backgrounds for hero, footer, and cards
- **Accessibility:** All interactive elements have visible focus, sufficient contrast, and semantic markup

## Usage

- Never use Evident or legacy branding
- All UI, docs, and assets must reference Evident
- Forensic, accessible, and trustworthy tone in all content
