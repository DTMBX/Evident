## ðŸŽ“ Usage Examples

### Example 1: Comprehensive Case Analysis
```python
import requests

# 1. Upload documents
files = {'file': open('complaint.pdf', 'rb')}
response = requests.post(
    'http://localhost:8000/api/v1/ediscovery-advanced/upload-document',
    files=files,
    data={'case_id': 'CASE-001', 'custodian': 'Plaintiff'}
)

# 2. Extract legal citations
response = requests.post(
    'http://localhost:8000/api/v1/research/extract-citations',
    json={'text': document_text, 'resolve': True}
)

# 3. Analyze for PII before disclosure
response = requests.post(
    'http://localhost:8000/api/v1/privacy/detect-pii',
    json={'text': document_text}
)

# 4. Generate timeline visualization
response = requests.post(
    'http://localhost:8000/api/v1/visualization/timeline',
    json={
        'events': case_events,
        'title': 'Case Timeline - Incident to Filing'
    }
)
```

### Example 2: BWC Footage Analysis
```python
# 1. Transcribe audio
response = requests.post(
    'http://localhost:8000/api/v1/av-forensics/transcribe',
    json={
        'audio_file_id': 'BWC001',
        'model_size': 'large',
        'include_timestamps': True
    }
)

# 2. Identify speakers
response = requests.post(
    'http://localhost:8000/api/v1/av-forensics/diarize-speakers',
    json={'audio_file_id': 'BWC001'}
)

# 3. Detect critical events
response = requests.post(
    'http://localhost:8000/api/v1/av-forensics/detect-audio-events',
    json={
        'audio_file_id': 'BWC001',
        'event_types': ['gunshot', 'scream', 'siren']
    }
)

# 4. Redact faces for privacy
response = requests.post(
    'http://localhost:8000/api/v1/privacy/redact-video',
    json={
        'video_id': 'BWC001',
        'redaction_options': {'blur_faces': True}
    }
)
```

### Example 3: Medical Record Analysis

# 1. Extract medical entities
response = requests.post(
