# Security Policy

## Reporting a Vulnerability

Please report security issues by email:

**security@Evident.info**

Include:

- a clear description of the issue
- affected file(s) / component(s)
- steps to reproduce (or proof-of-concept details if safe)
- impact assessment (what could happen)
- any suggested remediation

## Coordinated Disclosure

Please do not publicly disclose the issue until we respond and have had a
reasonable opportunity to address it.

## Supported Versions

This repository is a **preview**. There is no supported production release at
this time.
