## HIERARCHY PROTOCOL

**Title 4 USC Compliance:**

1. **US Flag Always Supreme**
   - Largest size
   - Leftmost position (position of honor)
   - Always visible
   - Never subordinate

2. **Other Flags Subordinate**
   - Smaller size (50px vs 60px)
   - Hidden by default
   - Revealed only by inquiry
   - Equal to each other (no hierarchy among subordinates)

3. **No Animations**
   - Panel appears/disappears instantly
   - No fade effects
   - No hover animations on flags
   - Static testimony only

4. **Proper Spacing**
   - 1rem gap between flags
   - Square alignment
   - Truthful colors (official hex codes)
