Endpoints

GET /api/analyses

List all analyses for the authenticated user

Query Parameters

  • page - Page number (default: 1)
  • per_page - Results per page (default: 20, max: 100)

Response Example

{ "total": 42, "page": 1, "per_page": 20, "analyses": [ { "id": "abc123", "filename": "BWC_2025-01-15.mp4", "case_number": "ATL-L-002794-25", "status": "completed", "created_at": "2025-01-15T14:30:00Z" } ] }
POST /api/upload

Upload a BWC video file

Request Body (multipart/form-data)

  • file - Video file (required)
  • case_number - Case identifier
  • acquired_by - Person who obtained evidence

Response Example

{ "upload_id": "xyz789", "filename": "BWC_2025-01-15.mp4", "file_hash": "sha256:abcd1234...", "file_size_mb": 145.6 }
POST /api/analyze

Start analysis on uploaded video

Request Body (JSON)

{ "upload_id": "xyz789", "case_number": "ATL-L-002794-25", "acquired_by": "John Smith", "source": "OPRA Request #2025-001" }
GET /api/analysis/{id}

Get analysis details and results

GET /api/analysis/{id}/report/{format}

Download analysis report

Formats

  • json - Structured JSON data
  • txt - Plain text report
  • md - Markdown report
