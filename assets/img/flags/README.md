## Image Specifications

- **Format:** PNG (preferred) or high-quality JPG
- **Resolution:** Minimum 800x600px, aspect ratio 4:3
- **Color space:** sRGB
- **File size:** Keep under 200KB per image for web performance
- **Background:** Transparent PNG recommended for flags

## License Verification

All images must be:

- U.S. Government public domain, OR
- Creative Commons CC0/Public Domain, OR
- Historical reproductions with verified public domain status

## Conversion from SVG to PNG

If downloading SVG files from Wikimedia Commons:

```bash
# Using ImageMagick or similar tool
convert input.svg -resize 800x600 -background transparent output.png
```
